== Installation ==

1. Upload plugin folder `google-analytics-client` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup your ad widget through the 'Themes/Widgets' menu in WordPress

== Frequently Asked Questions ==

= How can i get Google Analytics API Credentials ? =
To get Credentials like Client ID, Client Email and private key you should visit Google Analytics API from https://code.google.com/apis/console 
In API Access create new or use existing service account. 
Add your google service account email address to supported profiles as new user with minimal role User. 
Or choice existing account to connect with this profile what is recomended. 
You have to wait few minutes to synchonizing account status. 
Select your site profile and start collecting data. 
https://www.google.com/analytics/web/#management/Profile
